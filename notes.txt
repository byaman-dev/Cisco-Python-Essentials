# Cisco Networking Academy: Python Essentials 1
## Module 1: Introduction to Python and Computer Programming

Completed Date: October 2023
Summary: This module covered the fundamentals of how computers work, the role of programming languages, and the history of Python.

### Key Concepts Learned:
1. Programming Fundamentals:
   - Natural vs. Programming Languages.
   - High-level vs. Low-level languages.
   - The role of the Compiler (translates whole code at once) vs. the Interpreter (executes code line-by-line). Python is an interpreted language.

2. About Python:
   - Created by Guido van Rossum in 1991.
   - Named after "Monty Python’s Flying Circus."
   - Known for being easy to read, portable, and open-source.

3. Python Implementations:
   - CPython: The default and most popular implementation.
   - Jython: Python for Java.
   - PyPy: Focuses on speed.

4. The Compilation Process:
   - Source Code (.py) -> Bytecode (.pyc) -> Python Virtual Machine (PVM) -> Execution.

### Vocabulary:
- Syntax: The rules of the language.
- Semantics: The meaning of the code.
- PEP: Python Enhancement Proposal (specifically PEP 8 for style).
